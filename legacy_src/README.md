# NorthernVision CBO Website

## Overview
This repository hosts the website for **NorthernVision CBO**, an organization dedicated to empowering Northern Kenya through sustainable agriculture, education, and climate action.

## Migration Status
As of December 1, 2025, the legacy website (HTML/Bootstrap) has been moved to the `legacy_src/` directory. The root directory is currently being prepared for a modern reimplementation of the site.

### Legacy Code
The original source code can be found in:
- `legacy_src/index.html`: Main landing page
- `legacy_src/assets/`: Images, CSS, and JS files
